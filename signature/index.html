How to Add Your Signature

Outlook Desktop (Windows)

  1. Fill in your information above and click Copy Signature
  2. Open Outlook → File → Options → Mail → Signatures
  3. Click New, name it "Werco", then paste (Ctrl+V) into the edit box
  4. Set it as the default for new messages and replies
  5. Click OK to save

Outlook on the Web (Office 365)

  1. Fill in your information above and click Copy Signature
  2. Click the Settings gear (top right) → View all Outlook settings
  3. Go to Mail → Compose and reply
  4. Under Email signature, paste (Ctrl+V) into the editor
  5. Check the boxes for new messages and replies, then click Save

Outlook for Mac

  1. Fill in your information above and click Copy Signature
  2. Open Outlook → Outlook menu → Preferences → Signatures
  3. Click + to add a new signature, name it "Werco"
  4. Paste (Cmd+V) into the edit box and set as default
